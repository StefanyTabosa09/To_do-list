* To Do List

- Aplicação web de lista de tarefas desenvolvida com HTML, CSS e JavaScript, com foco em organização, interatividade e manipulação do DOM.

* Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Organização visual das atividades

* Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

* Objetivo do projeto

- Praticar conceitos fundamentais de desenvolvimento front-end, como estruturação de páginas, estilização e interatividade com JavaScript.
